# gpt-output-cleaner
Simple single page app that cleans GPT output. 

What it does:
- Strips out additional spaces and section breaks
- Removes uneccssary html

How to use:
1. Open the html file in a browswer
2. Paste in your copied GPT content
3. Choose formtting option if you like default or compact
4. Copy the output and paste into a doc
